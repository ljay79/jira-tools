Deprecation Notice! - Jira On-Premise only!

I am sorry to inform any users with a on-premise Jira Data Center
or Jira Server edition; this project is ending support for any on-premise Jira editions!

Due to lack of time and lack of a own on-premise Jira instances, i am unable to develop any support for such editions. They differ to much from Jira Cloud API versions and become more and more outdated, while Atlassian Jira Cloud is continously updating the cloud API endpoints.

I feel sorry for anyone using my add-on in past and is soon not able to anymore. I dont take this decision easily, but i feel unable to develop blindly on any non-cloud Jira API versions.

Thank you!

Thank you for using this add-on in the past.
Jira Cloud will continue to be supported.
